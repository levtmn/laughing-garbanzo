# laughing-garbanzo
Minimal secure Telegram bot for provisioning and managing Telemt users through the Telemt HTTP API.

The bot is intended for small private deployments where a limited set of trusted Telegram users can create, update, extend, disable, and inspect Telemt accounts. It is deliberately conservative:

- Telegram is the only user-facing interface
- Telemt API is not exposed publicly
- only allowlisted Telegram user IDs can use privileged commands
- fixed plans are preferred over arbitrary user-supplied limits
- secrets and full proxy links are never written to logs

## Goals

- create a Telemt user if it does not exist
- update an existing Telemt user if it already exists
- return a ready Telegram proxy link generated by Telemt
- keep the codebase small, auditable, and easy to operate
- для того чтобы пользователь получил ссылку нужно требовать оплаты через СБП на номер телефона в размере 100р - этого будет хватать на 1 месяц - подтверждение получения фин. средств будет проверятся вручную администратором

## Non-goals

This project does not aim to provide:

- public self-service signup
- multi-tenant administration
- arbitrary raw API access from chat
- direct public exposure of the Telemt API
- secret rotation from chat unless explicitly implemented and reviewed
